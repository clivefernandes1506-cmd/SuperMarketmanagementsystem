# Supermarket Management System

 Overview

A Windows Forms desktop application developed in C# using Entity
Framework Core and SQL Server LocalDB.

 Features

-   Product Management (CRUD)
-   Category Management (CRUD)
-   Supplier Management (CRUD)
-   Stock Management
-   Sales Recording
-   Product Search using a Binary Search Tree
-   Low Stock Alerts

 Technologies

-   C#
-   .NET Windows Forms
-   Entity Framework Core
-   SQL Server LocalDB
-   Visual Studio 2022

 Database

Tables: - Products - Categories - Suppliers - Stocks - Sales - SaleItems

 Running the Project

1.  Open the solution in Visual Studio.
2.  Restore NuGet packages if prompted.
3.  Ensure SQL Server LocalDB is installed.
4.  Update the connection string if required.
5.  Build and run the project.

 Project Structure

-   Forms
-   Models
-   Services
-   Data
-   DataStructures
